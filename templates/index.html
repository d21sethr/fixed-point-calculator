Overview of Fixed-Point Iteration

The fixed-point iteration method is an open root-finding methodology used to identify numerical approximations of a real root for a given equation $f(x) = 0$. The fundamental strategy involves transforming the equation algebraically into an equivalent, rearranged format:

$$x = g(x)$$

A value $\alpha$ that satisfies this relation is designated as a fixed point of the mapping function $g(x)$. By formulating an initial guess $x_0$, we define successive numerical approximations using the recurrence relation:

$$x_{n+1} = g(x_n), \quad n = 0, 1, 2, \dots$$

Convergence Criteria (The Fixed-Point Theorem)

Let $g(x)$ be a continuously differentiable function on an interval $[a, b]$ containing a fixed point $\alpha$. If $|g'(x)| \le k < 1$ for all $x \in [a, b]$, then the sequence generated by the iterations will converge to the unique root $\alpha$ for any initial choice $x_0 \in [a, b]$.

  • If $|g'(\alpha)| < 1$, the scheme converges.
  • If $|g'(\alpha)| > 1$, the scheme diverges from the root.
  • If $g'(\alpha)$ is positive, the trajectory paths converge monotonically.
  • If $g'(\alpha)$ is negative, the trajectory paths converge in an oscillating spiral fashion (cobweb pattern).

Fully Worked Numerical Examples

These static examples detail intermediate steps to illustrate the performance and computation flow of the method.

Example 1: Non-linear Algebraic Equation

Solve the algebraic equation: $f(x) = x^3 - x - 1 = 0$ near $x = 1.3$ to a tolerance of $\epsilon = 10^{-3}$.

1. Transformation to $x = g(x)$

We rearrange $x^3 - x - 1 = 0$ as $x^3 = x + 1$, yielding:

$$g(x) = (x + 1)^{1/3}$$
2. Analytical Convergence Check

Evaluate the derivative $g'(x)$ to check stability near the root:

$$g'(x) = \frac{1}{3}(x + 1)^{-2/3} = \frac{1}{3\sqrt[3]{(x+1)^2}}$$

For $x_0 = 1.5$:

$$|g'(1.5)| = \frac{1}{3\sqrt[3]{(2.5)^2}} \approx 0.1804 < 1$$

Since the magnitude is significantly smaller than $1$, stable convergence is guaranteed.

3. Iteration Sequence Steps
  • Step 0: Initial guess: $x_0 = 1.50000$
  • Step 1: $x_1 = g(1.50000) = (1.5 + 1)^{1/3} = (2.5)^{1/3} \approx \mathbf{1.35721}$
    Error: $|x_1 - x_0| = |1.35721 - 1.50000| = 0.14279$
  • Step 2: $x_2 = g(1.35721) = (1.35721 + 1)^{1/3} \approx \mathbf{1.33086}$
    Error: $|x_2 - x_1| = |1.33086 - 1.35721| = 0.02635$
  • Step 3: $x_3 = g(1.33086) = (1.33086 + 1)^{1/3} \approx \mathbf{1.32588}$
    Error: $|x_3 - x_2| = |1.32588 - 1.33086| = 0.00498$
  • Step 4: $x_4 = g(1.32588) = (1.32588 + 1)^{1/3} \approx \mathbf{1.32494}$
    Error: $|x_4 - x_3| = |1.32494 - 1.32588| = 0.00094 < 0.001$

Convergence reached at Step 4. The approximated root is $1.32494$.

Example 2: Transcendental Equation

Solve the transcendental equation: $f(x) = x - \cos(x) = 0$ using an initial guess $x_0 = 0.5$ to a tolerance of $\epsilon = 10^{-3}$.

1. Transformation to $x = g(x)$

By inspection, we isolate $x$ to get:

$$g(x) = \cos(x)$$
2. Analytical Convergence Check

Find the derivative $g'(x)$:

$$g'(x) = -\sin(x)$$

At $x_0 = 0.5$:

$$|g'(0.5)| = |-\sin(0.5)| \approx 0.47943 < 1$$

Since the value is less than $1$, the sequence will converge (expecting oscillatory convergence due to the negative derivative value near the actual intersection point).

3. Iteration Sequence Steps
  • Step 0: Initial guess: $x_0 = 0.50000$
  • Step 1: $x_1 = \cos(0.50000) \approx \mathbf{0.87758}$
    Error: $|x_1 - x_0| = |0.87758 - 0.50000| = 0.37758$
  • Step 2: $x_2 = \cos(0.87758) \approx \mathbf{0.63901}$
    Error: $|x_2 - x_1| = |0.63901 - 0.87758| = 0.23857$
  • Step 3: $x_3 = \cos(0.63901) \approx \mathbf{0.80269}$
    Error: $|x_3 - x_2| = |0.80269 - 0.63901| = 0.16368$
  • Step 4: $x_4 = \cos(0.80269) \approx \mathbf{0.69478}$
    Error: $|x_4 - x_3| = |0.69478 - 0.80269| = 0.10791$
  • Step 5: $x_5 = \cos(0.69478) \approx \mathbf{0.76820}$
    Error: $|x_5 - x_4| = 0.07342$

Note: Continuing this process yields values oscillating and converging toward the Dottie number ($x \approx 0.73908$).
